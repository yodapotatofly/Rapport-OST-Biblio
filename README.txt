Pour l'édition du rapport, je vous conseille d'utiliser TeXmaker, mais n'importe quel éditeur de texte fait l'affaire.

Pour la compilation, je vous conseille d'utiliser le make que je vous ai fourni.

Pour l'édition des références (citations qu'on vous reproche souvent, nous les méchants enseignants), je vous conseille d'utiliser JabRef, mais n'importe quel éditeur de texte fait l'affaire.

Pour compiler, rendez-vous dans le repertoire où se trouve votre fichier "rapport.tex", et lancer la commande "make" (sans guillemets).

Bon courage !
